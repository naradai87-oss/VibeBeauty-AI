# VibeBeauty AI ✨
> **AI-Driven Personal Stylist & Premium Beauty Experience**

VibeBeauty AI는 사용자의 고유한 분위기를 발견하고 최적의 스타일을 제안하는 프리미엄 AI 스타일리스트 플랫폼입니다. 초정밀 컬러 분석과 실루엣 측정을 통해 당신만을 위한 럭셔리 스타일링 경험을 제공합니다.

## 🌟 Key Features

- **AI 컬러 분석 (Color Analysis)**: ITA 지수 기반의 초정밀 피부톤 측정과 계절별 퍼스널 컬러 진단.
- **바디 쉐이핑 (Body Shaping)**: 전신 실루엣 분석을 통한 장점 극대화 코디네이션 및 스타일 제안.
- **프리미엄 큐레이션 (Premium Curation)**: 럭셔리 브랜드 및 쿠팡 파트너스 연계 최저가 맞춤형 추천.
- **스타일 로그 (Style Log)**: AI 분석 결과를 지브리 감성의 화보 카드로 저장 및 공유.

## 🛠 Tech Stack

- **Framework**: [Next.js 15 (App Router)](https://nextjs.org/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Animations**: [Motion (Framer Motion)](https://motion.dev/)
- **Backend/Auth**: [Supabase](https://supabase.com/)
- **Icons**: [Lucide React](https://lucide.dev/)
- **Export**: [html2canvas](https://html2canvas.hertzen.com/)

## 🚀 Getting Started

### Prerequisites
- Node.js 18+ 
- Supabase Project

### Installation
1. Repository 클론:
   ```bash
   git clone https://github.com/naradai87-oss/VibeBeauty-AI.git
   ```
2. 의존성 설치:
   ```bash
   npm install
   ```
3. 환경 변수 설정: `.env.local` 파일을 생성하고 다음 내용을 입력합니다.
   ```env
   NEXT_PUBLIC_SUPABASE_URL=your_supabase_url
   NEXT_PUBLIC_SUPABASE_ANON_KEY=your_supabase_anon_key
   NEXT_PUBLIC_COUPANG_PARTNERS_ID=your_id
   ```
4. 개발 서버 실행:
   ```bash
   npm run dev
   ```
